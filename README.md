# 실행 방법

## 1. 프로젝트 다운로드
```shell
npm install
```

## 2. 프론트 프로젝트 실행
```shell
npm start
```
### [locallhost:3000](http://localhost:3000)

## 3. 백엔드 프로젝트 실행
```shell
node server/index.js
```
